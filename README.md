## Features

- No server required
- Portable program
- Cross platform


## Tech Stack

**Client:** Blazor, .NET 8

**Server:** Blazor, .NET 8

**Database:** SQL Server


## Run Locally

Clone the project

```bash
  git clone https://github.com/AngeloDotNet/SagreEventi.SQLServer-NET8.git
```

Go to the project directory

```bash
  cd src\SagreEventi.Web.Server
```

Build the project

```bash
  dotnet build
```

Start the server

```bash
  dotnet run
```
